# Face Investigator Web

Versão web do Face Investigator para Streamlit Cloud.

## Como usar

1. Envie uma imagem alvo.
2. Envie várias imagens de comparação.
3. Clique em **Comparar**.
4. Veja os resultados e baixe o CSV.

## Requisitos

- Streamlit Cloud.
- Arquivo principal: `app.py`.
- Dependências em `requirements.txt`.
